# Writing Discipline

## Overview

Signal over decoration. Signal is content that carries meaning. Decoration is formatting that makes something look structured without adding information. If removing a formatting element loses no information, remove it. Apply this before writing — not as an editing pass after.

## What to Avoid

Avoid decorative dividers (repeated dashes, equals signs, or asterisks used as visual breaks). Avoid nested bullets beyond one level. Avoid bold used for visual weight rather than to mark terms so critical that missing them breaks understanding. Avoid headers on content too short to need navigation — if a section is one or two sentences, write prose instead. Avoid redundant parenthetical labels like "(unchanged)" or "(see above)" — if the information matters, state it; if it doesn't, cut it. Avoid listing properties as sub-bullets when they fit on one line.

## When Structure Is Earned

Use bullets when you have three or more genuinely parallel, discrete items with no natural prose flow. Use headers when content is long enough that a reader needs to navigate or skip sections. Bold only terms so critical that missing them breaks understanding. Use tables only when the grid itself is the content — when comparing items across two or more dimensions and the relationship between cells matters. IMPORTANT and NOTE callouts are appropriate for genuinely critical information the reader must not miss.

## Output

No structured output. This skill shapes how prose is written.

## Hard Stop

None. Guidance is internalized, not a gate.
